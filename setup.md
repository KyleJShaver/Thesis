## Install Dependencies
### linux: 
`sudo apt-get install -y git python3 python3-pip libsm6 libxrender1 libfontconfig1 libxtst6`

`git clone https://github.com/KyleJShaver/Thesis.git`

`cd Thesis`

`pip install -r requirements.txt`
